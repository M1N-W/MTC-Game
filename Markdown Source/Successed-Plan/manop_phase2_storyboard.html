Phase 1 — ครูผู้อดทน (100–51%)
⚡ 50% HP Trigger
Phase 2 — ห้องสอบมรณะ (50–0%)
1
Transition moment — animation beat
0.0s ครูมานพหยุดนิ่งทันที ทุก movement/attack หยุดชะงัก
0.3s กระดานดำข้างหลังแตกร้าวเป็นเส้น ๆ แล้วสว่างวาบเป็นสีแดง
0.8s แว่นตาของครูแสงสะท้อน — กลบตา 1 frame (hitFlash variant: white)
1.2s ครูก้มหัวช้า ๆ → เงยขึ้นอีกครั้ง — พร้อม dialogue
การหยุดนิ่ง 1+ วินาทีก่อนพูด = psychological pressure ที่ได้ผลกว่า roar ทันที — ผู้เล่นไม่รู้ว่าจะเกิดอะไร
2
Dialogue — voice bubble (UIManager)
บรรทัดหลัก (แสดงใน voice bubble ขนาดใหญ่):
"คะแนนสะสม: ศูนย์  ·  เวลาที่เหลือ: ไม่มี  ·  ความน่าจะเป็นที่จะรอด... คำนวณเองแล้วกัน"
"Score: zero · Time remaining: none · Probability of survival... calculate it yourself."
บรรทัดรอง (ตามมาหลัง 1.5s หรือ typewriter effect):
"บทเรียนสุดท้ายไม่มีตัวเลือก B"
ทำไมบรรทัดนี้? ใช้ format รายงานคะแนน (stat readout) ซึ่งเป็นสิ่งที่ผู้เล่นเห็นตลอดเกม — แต่ตอนนี้ครูพูดมันเป็นคำสาป ไม่ใช่คำเตือน ให้ความรู้สึก "เกมรู้จักคุณ"
3
Environment shift — ห้องเรียน → ห้องสอบมรณะ
แสงไฟฟลูออเรสเซนต์กะพริบ 3 ครั้ง → เปลี่ยนเป็นสีแดงคลุมห้อง (overlay rgba #8B0000 ~0.25)
กระดานดำเต็มไปด้วยสมการเรืองแสง — scrolling slow ตลอด Phase 2
หน้าต่างทั้งหมดสลับ sprite เป็น "มีลูกกรงเหล็ก" — ออกไม่ได้
โต๊ะนักเรียนในฉากหลัง animate เลื่อนเป็นแถวเข้าหาผู้เล่น (cosmetic only)
นาฬิกาในห้องปรากฏ — เดินถอยหลัง (visual pressure, ไม่ใช่ mechanic จริง)
BGM crossfade → track Phase 2: tempo +15%, key shift minor, เพิ่มเสียง chalk scraping loop
สมการบนกระดาน (chalk style, scrolling):
P(survive) = 0    lim(HP→0) = ∞ pain    Δt = -∞
∫₀ᵀ damage(t)dt ≫ maxHP    ∀ player: dead = true
ทุก visual change ต้องเป็น overlay/sprite swap บน map.js — ไม่แตะ collision ของห้อง
4
Phase 2 mechanics — attack patterns
Final Exam Barrage
ข้อสอบปรนัย (A-D)
ยิง projectile 4 ชุดพร้อมกัน เป็นรูปตัว A B C D — spread angle ต่างกัน ผู้เล่นต้อง "เลือก" ช่องว่างที่จะวิ่งผ่าน HIGH PRESSURE
Red Pen Correction
ขีดคะแนน
เส้นสีแดงกวาดข้ามห้องในแนวนอน 3 เส้นพร้อมกัน เตือนด้วย chalk-white flash 0.5s ก่อน ดาเมจสูง แต่หลบได้ด้วยการกระโดดช่องว่าง
Pop Quiz Summon
สอบย่อย
ครูเรียก "นักเรียน" (minion wave เล็ก) ออกมาแบบ deterministic — ตาม WAVE_SCHEDULE pattern แต่ frequency สูงกว่า Phase 1 x1.5
Zero Score Judgment
ได้ศูนย์คะแนน
AOE ระเบิดที่ตำแหน่งผู้เล่น — delay 1.2s พร้อม countdown circle บนพื้น (เลข 3-2-1 ลดลงใน circle) ถ้าไม่วิ่งออก = stun + damage
Enrage mechanic: ทุก 25% HP ที่หาย ใน Phase 2 (50%→25%→0%) ครูพูด short line: "ยังไม่พอ..." / "ห้ามออกห้องสอบ!" — และ attack speed ขึ้น +10%
